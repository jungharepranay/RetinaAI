Abstract: This paper presents RetinAI, a hybrid retinal disease screening framework that combines a three-model deep learning ensemble (EfficientNet-B3, DenseNet-121, ConvNeXt-Tiny) with classical computer vision-based biomarker extraction and rule-based clinical validation for multi-label classification on the ODIR-5K dataset (7,439 fundus images, eight diagnostic categories). Individual model outputs are fused through per-class AUC-weighted soft voting, and per-class decision thresholds are optimised via grid search to maximise F1-score independently for each condition. The ensemble achieves a macro AUC-ROC of 0.8600, macro F1-score of 0.6280, and Hamming Loss of 0.1439. A parallel feature extraction module computes 17 clinically grounded biomarkers, which are cross-referenced against predictions using rules derived from ETDRS, AAO, and Keith-Wagener-Barker guidelines. Grad-CAM heatmaps provide spatial explainability, and an interactive patient questionnaire contextualises findings without altering model confidence scores. The system is deployed as a FastAPI web application generating structured PDF screening reports.

Keywords: Retinal Disease Detection, Multi-label Classification, Ensemble Deep Learning, EfficientNet-B3, DenseNet-121, ConvNeXt-Tiny, ODIR-5K Dataset, Grad-CAM Explainability, Clinical Decision Support, Rule-Based Validation, Fundus Image Analysis
